Add components to OpenShift¶

Ansible to configure https://try.openshift.com[OpenShift] cluster for Red Hat Developer Workshops and Demos.

The role can also be used to install and configure:

  • [x] Red Hat OpenShift Serverless both Serving and Eventing

  • [x] Red Hat OpenShift Pipelines

  • [x] Argo CD

  • [x] Red Hat OpenShift Service Mesh

  • [x] Red Hat AMQ Streams

  • [x] [Red Hat Advanced Cluster Management for Kubernetes] (https://www.redhat.com/en/technologies/management/advanced-cluster-management)

  • [x] Apache Camel-K

  • [ ] Red Hat Code Ready Workspaces

Example Playbooks¶

The examples directory has various playbook examples to get started using this role:

e.g.

If you want to deploy OpenShift Serverless with Pipelines run:

ansible-playbook examples/serverless_pipelines.yml

If you don’t have Ansible installed locally, you can use the project OpenShift Demos Ansible EE to run the playbooks using Docker and Ansible Runner.
