Kamesh's Ansible Collections for OpenShift Demos
=================================================

This repo contains the `kameshsampath.demos` Ansible Collection. The collection includes many modules and roles that are used as part of my `OpenShift <https://try.openshift.com>`_ Demos. The roles and modules helps you setup and configure OpenShift resources and other underlying Cloud Provider resources.
